# Post-hoc Grid Heatmap

Functions for creating grid-based heatmaps of behavioral states and metrics.

## Overview

This module provides tools to:

- Create spatial heatmaps on grid-based representations
- Visualize state probabilities across maze locations
- Generate multi-panel heatmap displays
- Compare spatial patterns between groups
