# CoMPASS Level 1 - Data Preparation

Functions for preparing behavioral data for Level 1 Hidden Markov Model analysis.

## Overview

This module provides utilities to:

- Compute step lengths and turn angles from position data
- Handle geographic and Euclidean coordinate systems
- Calculate distances and angles to target locations
- Prepare data in the format required for HMM fitting
